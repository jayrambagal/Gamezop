## Setup

**Clone the project**
`git clone git@github.com:jayrambagal/gamezop.git`

**Change the directory**
`cd gamezop`

**Add `.env` file**
`API_KEY = gamezop_api`

**Install the dependencies**
`npm install`

**The command to be run to start the server and other setups**
`npm run dev`
